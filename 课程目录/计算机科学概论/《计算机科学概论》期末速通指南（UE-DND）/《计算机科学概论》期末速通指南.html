3. 冯·诺依曼体系结构

  1. 控制单元 (Control Unit)：

    负责解释和执行存储在内存中的指令，并控制其他组件的操作。

  1. 算术逻辑单元 (ALU)：

    负责执行算术运算（如加法、减法）和逻辑运算（如比较大小、与或非操作）。

  1. 存储器 (Memory)：

    用于存储数据和程序。在冯·诺依曼架构中，内存中同时存储指令和数据。

  1. 输入设备 (Input Devices)：

    用于向计算机输入数据或指令，例如键盘、鼠标等。

  1. 输出设备 (Output Devices)：

    用于将计算结果或信息输出到外部设备，例如显示器、打印机等。

3. 进制转换

  • 二进制与十进制
    • 二进制转十进制：将二进制数按位权值展开，再进行求和。
      • 位权值展开后为以下结构

        将要展开的数各个位依次放在乘号左边，乘号右边的数字为要展开的数的进制。

        上标 m=m = m=﻿ 小数点前位数−1 -1−1﻿；n=n = n=﻿ 小数点后位数。

    • 十进制转二进制：将十进制分解为整数部分和小数部分，对两部分分别进行转换，然后相加得到结果。
      • 整数部分

        转换的规则为：

        1️⃣ 对十进制数使用短除法（除以二进制的基数 222﻿），将商数继续短除直至不能再除。

        2️⃣ 将所有短除后的余数，作为二进制的结果并逆序排列，即为转换后的二进制数。

        例如，将十进制整数 46 4646﻿ 转换为二进制整数：

        除数商余数
        46230
        23111
        1151
        521
        210
        101

        转换后结果为 101110101110 101110﻿，注意是逆序排列。

      • 小数部分

        转换的规则为：

        1️⃣ 将十进制小数乘以二进制的基数 222﻿，然后将得到的积的小数点前的数（0或1）作为二进制的结果，并正序排列，直到积的小数部分为0。

        例如，将十进制小数 0.375 0.375 0.375﻿ 转换为二进制小数：

        乘数积整数
        0.3750.750
        0.751.51
        0.51.01

        转换后结果为 0.0110.011 0.011﻿，注意是正序排列。

  • 二进制与八进制
    • 二进制转八进制：将二进制分解为整数部分和小数部分，对两部分分别进行转换，然后相加得到结果。
      • 整数部分

        转换的规则为：

        1️⃣ 从右向左，每三位一组进行分组。

        2️⃣ 如果最左边不足三位，可以在左边补零。

        3️⃣ 将每组三位二进制数转换为对应的八进制数字，方法与二进制转十进制相同。

      • 小数部分

        转换的规则为：

        1️⃣ 从左向右，每三位一组进行分组。

        2️⃣ 如果最右边不足三位，可以在右边补零。

        3️⃣ 将每组三位二进制数转换为对应的八进制数字，方法与二进制转十进制相同。

    • 八进制转二进制：八进制的每一位可以直接转换为三位二进制数，方法与十进制转二进制相同（短除）。
  • 二进制与十六进制
    • 二进制转十六进制：与以上转换类似，不同的是每四位一组进行分组。

4. 信息的编码

  • 整数的编码
    • 计算机中同一类型的数据具有相同的长度，与数据的实际长度无关。

      假设某计算机系统中整数占2字节（即16位二进制），则所有整数的长度都是16位。

    • 在计算机二进制数的表示中，原码、反码和补码是表示有符号整数的三种方式。

      它们主要用于处理正负数的表示。

      • 原码：最简单的有符号数表示方法。使用二进制的最高位（最左边第一位）表示符号，其余位表示数值大小。

        规则：
        符号位：最高位（最左边位）表示符号，0 表示正数，1 表示负数。
        数值部分：其余位表示该数的绝对值。

      • 反码

        规则：
        正数的反码与原码相同。
        负数的反码：符号位不变，
        数值部分（并非符号位）按位取反（ 0 变 1，1 变0 ）。

      • 补码

        规则：
        正数的补码与原码相同。
        负数的补码：将负数的原码的
        数值部分（并非符号位）按位取反，然后加 1（即：补码 = 反码 + 1）。
        补码转原码步骤相同（反码 + 1），不要反向操作，容易混淆。

        补码的优点是表示0的方式唯一，且方便进行算术运算。

  • 浮点数的编码

    浮点数：在日常生活中， 对于很大的数或很小的数， 必须用科学计数法来表示。小数点位置不固定， 称为浮点数。

    • IEEE754 标准表示法 （一种广泛使用的浮点数运算标准，也称余127码）
      以单精度格式为例，共用32位二进制来存储一个浮点数。
      其中，符号位占1位，阶码指数占8位，尾数占23位（无符号数）。

  • 数据溢出

    原因是要表示的数值，超过了系统能够表示的范围。

    • 用补码方式表示n位带符号整数时，表示范围为 −2n−1～2n−1−1 -2^{n-1} ～ 2^{n-1} - 1−2n−1～2n−1−1﻿。因此在计算加减法时，需要注意可能会出现的数据溢出情况。
    • 先将两个数直接相加，若结果超出了n位带符号整数的表示范围，则溢出。
    • 溢出后结果的简单计算方法（适用于溢出范围不大的情况）：

      正溢出的计算：(a1+a2)−256=result(a1 + a2) - 256 = result (a1+a2)−256=result﻿ 再将 resultresultresult﻿ 转换进制

      负溢出的计算：(a1+a2)+256=result(a1 + a2) + 256 = result (a1+a2)+256=result﻿ 再将 resultresultresult﻿ 转换进制

5. 逻辑电路

  • 门电路（基本）
    • 与门：输入 A 和 B 同时为1，输出为1，否则为0。
    • 或门：输入 A 和 B 中只要有一个为1，输出为1，否则为0。
    • 非门：输入A为1，输出为0，输入A为0，输出为1。
    • 异或门：输入A和B相同，输出为0，输入A和B不同，输出为1。
    • 与非门：先对两个输入进行与运算（输入A和B同时为1，输出为1，否则为0），然后将其结果取反。
    • 或非门：先对两个输入进行或运算（输入A和B中只要有一个为1，输出为1，否则为0），然后对结果取反。
  • 组合电路
    • 半加器
    • 全加器：考虑进位输入的加法电路
    • 时序电路
      • S-R锁存器

6. 计算机部件

  • 存储器
    • 地址空间： 所有在存储器中标识的独立的地址单元的总称。

      地址空间的计算：如果计算机内存有 NNN﻿ 个字的存储空间，那就需要有 log2(N)log2(N)log2(N)﻿ 位的二进制无符号整数来确定一个存储单元。

    • 内存储器
      • 只读存储器ROM(Read only memory)，又称非易失性存储器。
      • 随机存储器RAM(Random access memory)，又称易失性存储器。
    • 高速缓冲存储器(Cache)，简称缓存：介于内存和CPU之间。位置可以在CPU芯片的内部， 也可以在CPU芯片的外部。
  • CPU的构成
    • 总线
    • 运算器(ALU)：主要由算术逻辑运算部件和寄存器组成。
    • 控制器
      • 程序计数器(ProgramCounter)：存放下一条要执行的指令的地址。
      • 指令寄存器(IR)：存放当前正在执行的指令。
      • 指令译码器(ID)：对当前指令进行翻译。
      • 时序控制电路：提供计算机运行时间的节拍。
      • 微操作控制电路：产生各种控制信号。
  • 程序的执行过程
    • 指令：分为 操作码 和 操作数 两部分。
      • 指令编码的头部为操作码，中间的数为寄存器编号，最后的数为存储单元编号。

2. 算法(Algorithm)

  • 算法是对特定问题求解步骤的一种描述，是指令的有限序列。
  • 算法的特性
    • 输入：一个算法有 0 个或多个输入
    • 输入：一个算法有 0 个或多个输入
    • 有穷性：一个算法必须在执行有限的步骤之后结束
    • 确定性：算法的每一步骤都必须有确切的定义，对于相同的输入只能有相同的输出
    • 可行性：算法的每一步骤都必须是可行的，也就是说，每一步骤都能够通过执行有限次数完成

七、「 计算机通讯与网络 」

1. 计算机通信

  • 信号分成两大类：数字信号和模拟信号。
    • 数字信号是一系列的脉冲，脉冲的状态只有两种，例如高电平和低电平。
    • 模拟信号是一种连续变化的波，基本特征是频率和振幅。
  • 调制与解调
    • 把数字信号转换为模拟信号称为调制；把模拟信号转换为数字信号称为解调。
    • 完成调制和解调的设备叫做调制解调器， 也称为Modem（光猫）。
